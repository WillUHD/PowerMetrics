# PowerMetrics
An easy to use Intel Power Gadget style monitor. Quick features: 
- **Passwordless**: uses IOKit bindings for hardware reports instead of `powermetrics`
- **Lightweight**: uses the same or less nominal CPU % compared to Mx PG
- **Accurate**: the native power and frequency reporting without being scaled down by utilization like Mx PG does
- **Familiar**: same frequency band drawing style as Intel PG
- **Customizable**: can customize the polling rate and graph capacity
Supports M5-class chips natively (P/E/S clusters), with support for other generations coming. Mainly built for personal use!
